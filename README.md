## Install from source

Clone this repository, cd into it. Issue:

    npm install

cd into node_modules/coloredcoinjs-lib, issue:

    npm install && npm run compile

cd into the node_modules directory of coloredcoinjs-lib, and remove the bitcore lib:

    rm -rf bitcore/
